These classes act as a simplification of the Py\+Gtk (Py\+Gobject) classes.

For instance the Label takes a constructor, and feeds the arguments as one would expect. Like this\+: \begin{DoxyVerb}   def __init__(self, textm = "", widget = None, tooltip=None, font = None):
\end{DoxyVerb}
 The defaults are set to a reasonable value, and the named argument can be set on one line. This makes the code look compact and maintainable.

See descendent projects for examples. (pyedpro; pycal; pggui; ...)
